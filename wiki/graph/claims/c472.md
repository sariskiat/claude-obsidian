RDF enables native inference while Property Graphs are faster for traversal

> [!quote]
> LPGs (labeled property graphs, e.g. Neo4j) store nodes-with-properties connected by labeled edges — fast, intuitive, great for traversal-heavy work like dependency analysis. RDF represents everything as subject-predicate-object triples with formal logical semantics, enabling native inference.
