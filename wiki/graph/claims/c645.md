Existing MoE expert offloading suffers from high latency: loading experts for a single step requires up to 11.3B parameter transfer, incurring 0.7s latency on PCIe 4.0x16.

> each expert is 176M in size, and loading the necessary experts for a single decoding step would require up to 11.3B of parameter transfer. If offloading to CPU VRAM is selected, using a GPU with PCIe 4.0x16 would still incur a transfer latency of 0.7s per step.
